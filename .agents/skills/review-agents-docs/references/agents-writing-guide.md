## AGENTS.mdのレビュー観点

### 役割と範囲
- `AGENTS.md` には全作業で常に効く上位ルールだけを書く。
- 実装詳細を `AGENTS.md` に書きすぎない。
- 同じ主題のルールは一か所へ集約する。

### 記述品質
- ルールは1行1ルールで書く。
- 曖昧な条件語を避け、条件を明示する。
- 禁止事項は対象と条件を明示する。

### 整合性
- `docs/coding-standards.md` へ委譲すべき内容が `AGENTS.md` に混在していないことを確認する。
- 必須記載項目が欠けていないことを確認する。
